## Program Overview
The **MotorPH Payroll System** is a Java-based application designed to manage and calculate employee payroll. The system allows payroll staff to:  
- View employee information (ID, name, birthday)  
- Calculate weekly hours worked with a 10-minute grace period  
- Compute gross and net salaries, including mandatory deductions (SSS, PhilHealth, Pag-ibig, Withholding Tax)  
- Display payroll summaries for verification and reporting

## Features
- Employee data retrieval and display  
- Weekly work-hour calculation with grace period  
- Gross and net salary calculation with government deductions  
- Payroll summary presentation for verification  

## Non-Functional Highlights
- **Accuracy:** Correct calculations with zero rounding errors  
- **Usability:** Clear console-based output  
- **Reliability:** Handles missing or incorrect log-in/out data gracefully  
- **Maintainability:** Modular and well-documented code  
- **Performance:** Efficient processing (<2 seconds for employee list display)
